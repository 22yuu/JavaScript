## Development enviroment

- npm init -y
- yarn add -D json-server (-D는 개발모드 배포시 해당 패키지는 제외하고 배포된다.)
    - json-server : 서버없이 json 기반으로 데이터 통신을 할 수 있게 해주는 패키지
- db.json 파일 생성 -> 가상의 데이터베이스 역할
- Live Server 확장
