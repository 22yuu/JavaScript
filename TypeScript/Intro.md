# 타입스크립트 개요
- 오픈소스 프로그래밍 언어
- 자바스크립트의 상위 집합으로 ECMA스크립트의 최신 표준을 지원
- 정적인 언어로 컴팡리 시간에 타입을 검사
- 장점
    - 강력한 타입으로 대규모 애플리케이션 개발에 용이
    - 유명한 자바스크립트 라이브러리와의 편리한 사용
    - 개발 도구에서의 강력한 지원
